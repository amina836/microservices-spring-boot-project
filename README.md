# <u>SchoolSystem-Microservices</u>

**Microservices Project using Spring Boot & Spring Cloud

Built a distributed microservices system consisting of Student Service and School Service, demonstrating core microservices architecture principles.

Implemented PostgreSQL with JPA/Hibernate for persistent data storage and entity management.

Designed and developed RESTful APIs to handle CRUD operations across services.

Enabled inter-service communication using OpenFeign to achieve seamless service-to-service interaction.

**Ongoing Architecture Enhancements
Implementing Eureka Discovery Server for service registration and discovery
Integrating Spring Cloud Gateway as a centralized API Gateway for routing and request management
Centralizing configuration using Spring Cloud Config Server to manage distributed configurations efficiently
