-Microservices Project using Spring Boot
Built a distributed microservices system consisting of Student and School services.
Integrated PostgreSQL database with JPA/Hibernate for persistent storage.
Implemented inter-service communication using OpenFeign for seamless service interaction.
Designed and developed RESTful APIs for CRUD operations across services.

-Ongoing Enhancements (Microservices Architecture Expansion):

Implementing Eureka Discovery Server for service registration and discovery.
Integrating Spring Cloud Gateway as a centralized API Gateway.
Centralizing configuration management using Spring Cloud Config Serve
